Installation

To use this skin on AO3, you’ll be creating a small set of linked skins. The base file sets up variables and structure, but you’ll also need to apply one of the theme files (Dawn, Moon, Cool, etc.) to actually get colors.

1. Create the Base Skin

  • Go to your Dashboard → Skins → Create Site Skin.
  • Give it a name, e.g. [BBC] Rosé Pine - Base.
  • Paste the contents of base.css into the CSS field.
  • Under Advanced, set it to Parent Only.
  • Click Submit.

2. Create a Theme Skin (Required)

This step provides the actual colors. Pick one of the themes below.

  • Create a skin titled something like [BBC] Rosé Pine Dawn or [BBC] Rosé Pine Moon.
  • Paste the code from one of the theme files:
    • Rosé Pine Dawn
    • Rosé Pine Moon
    • Rosé Pine Moon Cool
    • Rosé Pine
    • Rosé Pine Cool
  • Set it to Parent Only and submit.
  • (Optional) To make this act as a dark mode: edit the skin, go to Advanced → Choose @media, and set it to (prefers-color-scheme: dark).

3. Create the Tablet Skin

  • Create a skin titled [BBC] Rosé Pine - Tablet.
  • Paste in tablet.css.
  • Under Advanced → Choose @media, select: only screen (max-width: 62em).
  • Set to Parent Only and submit.

4. Create the Mobile Skin

  • Create a skin titled [BBC] Rosé Pine - Mobile.
  • Paste in mobile.css.
  • Under Advanced → Choose @media, select: only screen (max-width: 42em).
  • Set to Parent Only and submit.

5. Chain Them Together

  • Create one final skin, e.g. [BBC] Rosé Pine Dawn - Default.
  • In the CSS field, paste .rose-pine {opacity: 1;} (this is just a placeholder so AO3 lets you save).
  • Under Advanced → Parent Skins, add the skins you created in this order:
    1. [BBC] Rosé Pine - Base
    2. [BBC] Rosé Pine Dawn / [BBC] Rosé Pine Moon / [BBC] Rosé Pine
    3. [BBC] Rosé Pine - Tablet
    4. [BBC] Rosé Pine - Mobile
  • Click Submit, then Use.

Customization

  • Fonts: recommended are Domine (serif) and Figtree (sans-serif). Code snippets can use Victor Mono. For the cursive version of the code font, just remove the comments from /* font-style: italic; */ in your theme file.
  • Colors: each theme’s variables are listed at the top of its CSS file. To tweak, open the file in a text editor, find the color you want to change, and use Ctrl+H to replace it everywhere.
  • Tag symbol: You can change the little symbol that appears after tags. Look for this rule at the end of the CSS: 
    a.tag:not(.fandom.index.group a.tag,
h2.heading a.tag,
.splash .favorite a.tag,
h5.fandoms.heading a.tag,
dd.fandom.tags a,
[class^=warning] a.tag,
[class^=relationship] a,
[class^=character] a.tag):after {
  content: "\00a0\00a0✿";
}
    Swap ✿ for whatever you like (♥, ✦, ɞ, etc.).
