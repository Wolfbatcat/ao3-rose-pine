Installation

To use this site skin, you'll need to create and chain several individual skins together.

1. Create the Base Skin

  • Go to your Dashboard → Skins → Create Site Skin.
  • Give it a unique title (e.g., "[BBC] Rosé Pine Dawn").
  • Paste the base CSS code into the CSS field.
  • Under Advanced, select Parent Only.
  • Click Submit.

2. Optional: Add a Color Variation

Skip this step if you're solely using the base Rosé Pine Dawn scheme.

  • Create another skin with a title like "[BBC] Rosé Pine Moon".
  • Paste the alternate color scheme CSS into the CSS field:
    • Rosé Pine Moon
    • Rosé Pine Moon Cool
    • Rosé Pine
    • Rosé Pine Cool
  • Set it to Parent Only and submit.

3. Create the Tablet Skin

  • Create a new skin titled "[BBC] Rosé Pine - Tablet".
  • Paste the tablet CSS code into the CSS field.
  • Under Advanced, go to Choose @media and select:
    • only screen (max-width: 62em)
  • Submit the skin.

4. Create the Mobile Skin

  • Create a new skin titled "[BBC] Rosé Pine - Mobile".
  • Paste the mobile CSS code into the CSS field.
  • Under Advanced, go to Choose @media and select:
    • only screen (max-width: 42em)
  • Submit the skin.

5. Chain All Skins Together

  • Create one final skin titled "[BBC] Rosé Pine - Default".
  • Paste the default skin code into the CSS field. (This contains font settings and is required to save the skin.)
  • Under Advanced, add all the skins you created, in this order:
    1. Base Skin
    2. Optional Color Skin (if used)
    3. Tablet Skin
    4. Mobile Skin
  • Click Submit, then Use.

Customization

  • For fonts, I highly recommended Domine (serif) and Outfit (sans-serif). If you use Outfit, you can also use Pier Sans for the headings.
  • Color hex codes are listed at the top of each CSS file for easy reference. To change accent colors:
    1. Paste the code into a text editor.
    2. Highlight the color you'd like to replace on the color guide.
    3. Press Ctrl+H and replace all instances with the color you want.
    4. NOTE: Using 'Replace All' will replace every instance of that color in the guide! If you simply want to change tag colors, navigate to the tag section and make the changes individually.
  • Enable Dark Mode Support
    • If you added an alternate color skin and want it to act as a dark mode: edit that skin and under Advanced → Choose @media, select: (prefers-color-scheme: dark)
    • Update the skin.
